Acerca de

Este sitio es un ejercicio que integra varias técnicas HTML y contenido temático de One Piece.

Fue creado como parte de una práctica para combinar videos, formularios, mapas de imagen, tablas accesibles, listas anidadas y navegación con iconos.

El objetivo es ofrecer contenido interesante y accesible para fans de One Piece.
